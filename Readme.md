pages.py 內容說明
----
+ dict name = `page_list`
+ page_list {頁面名稱: ['頁面路徑 or 點擊目標 Xpath', '檢查內容物的 Xpath', '跳頁類型(0/1)']}
+ 跳頁類型
  + 0: 直接轉頁 -> value[0] = 頁面路徑
  + 1: 使用 click() -> value[1] = 點擊目標 Xpath
